LDS Study Mate

A Google Docs add-on that generates easy to read scripture links
Get Started

LDS Study Mate lets you generate easy to read links to the scriptures at lds.org. You can type out the scripture reference or paste a link from Scripture Links. After a minute the bothe hand typed scripture references and pasted links will be converted into easy to ready, working links that will quickly take you to not only the right page but also the right locaiton on the page, with the correct verses highlighted as well.
